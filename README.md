# crypto_dashboard

get data from official binance API [https://binance-docs.github.io/apidocs/spot/en/]

## Server side install
❯ pip install -r requirements.txt

## Run Server
❯ uvicorn main:app --reload

or alternative command

❯ python -m uvicorn main:app --reload

## Client side install

❯ npm -ci

## Run Client

❯ npm start
